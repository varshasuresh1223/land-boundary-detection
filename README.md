# Land Boundary Detection using Graham Scan Algorithm

## 📌 Project Description
This project implements the Graham Scan Convex Hull Algorithm for detecting the outer boundary of land coordinate points.

GTK4 graphics library is used for visualization.

---

## 🚀 Features
- Convex Hull Generation
- Land Boundary Detection
- Graphical Visualization using GTK4
- Coordinate Labels
- Hull Point Highlighting
- Real-Time Simulation

---

## 🛠 Technologies Used
- C Programming
- GTK4
- GCC Compiler
- VS Code
- MSYS2 MinGW64

---

## 📷 Input
User enters coordinate points manually.

Example:
100 100  
200 300  
400 200  
500 400  

---

## 📷 Output
The program displays:
- Blue inner points
- Green convex hull points
- Red boundary lines

---

## 📈 Time Complexity
O(n log n)
